# Flutter Slider Demo

This project is a small Flutter app that demonstrates the Slider widget with a realistic music volume example.

## What it shows
- A slider with a realistic use case: adjusting music volume
- Three easy-to-explain properties:
  - min
  - max
  - divisions
- A live percentage display that updates as the slider moves

## Run locally
1. Install Flutter and ensure it is on your PATH.
2. From the project folder, run:
   ```bash
   flutter run
   ```
3. If you want to verify the widget test:
   ```bash
   flutter test
   ```

## Presentation points
You can explain the demo in 3–5 minutes:
- The Slider widget lets users select a value within a range.
- In this app, it controls the music volume.
- The min and max properties set the range.
- The divisions property creates equal steps.
- The label and live text show the current value.
